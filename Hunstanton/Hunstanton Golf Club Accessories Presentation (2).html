How To Use This Configurator

This page is designed to help you review accessory options clearly before sending your final selections back. You can build variations for each product, compare saved options, and then choose exactly what should appear in your final summary.

STEP 1

Design Each Item

Use the Bag Tag, Pouch and Sticker tabs to work through colours, text and layouts for each product.

STEP 2

Save Options

Whenever you create a version you like for a product, use the save button. This stores that product option in Compare so you can keep alternatives.

STEP 3

Use Compare

In the Compare tab, review saved options for each product and choose which Bag Tag, Pouch and Sticker option should be used in the final summary.

STEP 4

Check Summary

Once you are happy, go to Summary. That page shows the chosen selections and can be screenshotted, printed to PDF, or copied as text to send back.

Thank you for using the Arbor Golf accessory configurator. When your choices are complete, please send over the Summary page or the copied summary text so the final selections can be reviewed.
